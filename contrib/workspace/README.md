# Welcome to Ozwell Studio

This is your workspace directory. Build your full-stack web application here.

## Getting Started

Start your application on **port 3000** and it will be available in the
**Preview** tab of the Studio dashboard.

For example, with Node.js:

```bash
npx create-vite@latest . --template react && npm install && npm run dev -- --port 3000
```

Or with Python:

```bash
python3 -m http.server 3000
```
